The BPMN Workflow Engine

Many operational KPIs are taken here. Every application follows the same governed path — an eight-state lifecycle — with an SLA clock running on each state. BPMN stands for Business Process Model and Notation, an international standard for drawing and running step-by-step business processes.

The standards & shorthand you'll meet

A quick recall, so the abbreviations in the KPI stories are already familiar.

  • BPMN — Business Process Model and Notation: the case-lifecycle standard shown on the previous slide.
  • CMF — Common Message Format: the single shared data model every registry translates into.
  • Once-only — a verified fact is read once and reused; the citizen never re-submits it.
  • eID · eIDAS — the national digital identity, aligned to the European eIDAS standard.
  • WORM — Write Once, Read Many: tamper-proof storage for the seven-year audit trail.
  • DORA — the four delivery metrics (deploy frequency, lead time, change-failure rate, restore time) behind the DevOps KPIs.
  • API — Application Programming Interface: the defined way two software systems talk to each other.
